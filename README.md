# cea608-types

Repository containing an implementation parsing and writing the bytestream
defined in CTA-608 (formerly known as CEA-608).

## Relevant standards

 - [ANSI/CTA-608-E S-2019](https://shop.cta.tech/products/line-21-data-services).
